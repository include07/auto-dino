# Auto-Dino

## Introduction

This repository is part of the Gadz'IT Python workshop, it represents the final project of the course, which is Python script that automatically plays the offline dinosor game found via the chrome browser by visiting the following link: [dino-game](chrome://dino/).

## Ressources

1. A working Python installation, refer to [Python's official site](https://www.python.org/) for instructions on installing the Python interpreter .
2. A package manager, pip is the go to choice, refer to the [pip's documentation](https://pip.pypa.io/en/stable/installation/) for detailed instructions for your specific platform.
3. The Python package PyAutoGUI, refer to the [documentation](https://pyautogui.readthedocs.io/en/latest/install.html) for platform specific installation instructions.
